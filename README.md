## 🚀 Overview

FoodVision AI is an end-to-end computer vision application that transforms food images into actionable insights using deep learning and transfer learning techniques.

The system allows users to upload a food photograph and instantly receive predictions generated by a fine-tuned MobileNetV3 neural network trained on the Food-11 dataset.

Unlike traditional machine learning projects that remain confined to notebooks, FoodVision AI has been deployed as a publicly accessible web application, enabling real-world interaction through a custom domain and responsive interface.

## 🧠 Technical Highlights

### Deep Learning Backbone

FoodVision AI leverages:

* MobileNetV3 Small
* Transfer Learning
* PyTorch
* Computer Vision

The model benefits from feature representations learned from ImageNet-scale training, allowing efficient adaptation to food classification tasks while maintaining low inference latency.

### Dataset Scale

The system was trained using the Food-11 dataset containing:

* 16,000+ food images
* 11 food categories
* Diverse lighting conditions
* Real-world food photography

Categories include:

* Bread
* Dairy Product
* Dessert
* Egg
* Fried Food
* Meat
* Noodles / Pasta
* Rice
* Seafood
* Soup
* Vegetable / Fruit
